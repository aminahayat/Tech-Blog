# Tech-Blog

The purpose of this app is to create a tech 'talk space' where users can create an account, create a post, edit, delete, view and add comments to, using MYSQL, JS and MVC.

# Technologies Used
* Handlebars
* SQL
* Javascript
* CSS
* MVC

# Installation
* Start with cloning my repository
* npm install
* node server.js
* voila!
